# Changelog

本プロジェクトの変更履歴。Semantic Versioning に準拠する。

## [1.0.0] - 2026-05-10

初版リリース。

### 追加

- Chrome / Edge 拡張機能（Manifest V3）
  - 写真詳細画面でのタグ追加・削除（チップ UI）
  - 一覧サムネイルへのタグバッジ表示
  - タグでの絞り込み（OR 条件、複数選択可）
  - フィルター結果の全画面ギャラリー表示
  - 複数選択モードでの一括タグ付け
  - タグ管理画面（リネーム・統合・削除・件数集計）
  - サムネイルの自動 base64 キャッシュ
- Mobile PWA
  - Google Drive を介した PC との自動同期
  - タグから写真を検索する読み取り専用ビュー
  - サムネイル表示と日付による絞り込み
  - 写真クリックで Google フォト純正アプリへ遷移
  - サイレント再認証による永続ログイン
  - 複数 Google アカウント間の切り替え
- インフラ・配布
  - GitHub Actions による PWA の自動デプロイ
  - 非機密スコープのみ使用（OAuth 検証不要）
  - プライバシーポリシーページの同梱

### 設計上の特記事項

- バックエンドサーバーを持たない完全クライアントサイド構成
- すべてのユーザーデータは Google Drive `appDataFolder` に保存
- フレームワーク非依存（Vanilla JS / HTML / CSS のみ）
- ビルドステップ不要
