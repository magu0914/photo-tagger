Photo Tagger

PC で付けたタグから写真をすぐ探せるモバイルビューアです。

Google アカウントでサインインすると、タグ情報（Drive の appDataFolder に保存されたもの）を読み込みます。写真本体や他のファイルにはアクセスしません。

⚠️ 設定が必要です

config.js を作成して OAuth クライアント ID を設定してください。

window.PHOTO_TAGGER_CONFIG = {
  GOOGLE_CLIENT_ID: 'xxx.apps.googleusercontent.com'
};

詳細は 06_PWA_デプロイガイド.md 参照。
