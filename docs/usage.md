# 利用ガイド

PC とスマホでの普段の使い方をまとめる。

## 役割分担

PC（Chrome 拡張機能）でタグの作成・付与・管理を行い、スマホ（PWA）でタグから写真を検索する。Photos Library API の制約により、スマホ側からの新規タグ付けはできない。

```
PC                              スマホ
タグの作成・付与・管理   →     タグから写真を検索
                                ↓
                                Google フォト純正アプリで実物表示
```

## PC での使い方

### タグを付ける

1. `https://photos.google.com/` で写真を開く
2. 画面右下のオーバーレイにタグ入力欄が表示される
3. タグ名を入力 → Enter
4. チップが追加され、Drive に保存される

### 一括タグ付け

1. サムネイルにホバーすると左上に丸いチェックボタンが現れる。これをクリックして選択モードに入る
2. 複数の写真を選択する
3. 画面下部中央に「N 件選択中」のバーが表示される
4. タグ名を入力 → Enter で全選択写真に一括付与

日付ヘッダの「N の写真をすべて選択」も使える。

### タグでフィルター

一覧画面でオーバーレイのフィルターチップをクリックすると、該当しない写真が薄く表示される。複数選択は OR 条件。

### 結果を全画面ギャラリーで見る

フィルター適用中、オーバーレイの「結果を一覧で見る (N 件)」ボタンを押すと、該当全件をサムネイル付きで全画面表示する。仮想スクロールに依存しないので、万単位の写真でも実用的に動く。

### タグの管理

`chrome://extensions/` → Photo Tagger の「詳細」→「拡張機能のオプション」を開くと、タグの一覧・検索・ソート・リネーム・統合・削除ができる管理画面が開く。

## スマホでの使い方

### 初回セットアップ

1. PWA の公開 URL をスマホブラウザで開く
2. 「Google でサインイン」→ アカウント選択 → 同意

### ホーム画面に追加

ネイティブアプリ風の起動感にするため、ホーム画面にショートカットを置く。

iOS Safari の場合：共有ボタン → 「ホーム画面に追加」

Android Chrome の場合：右上の三点メニュー → 「ホーム画面に追加」または「アプリをインストール」

### タグから写真を探す

1. ホーム画面のアイコンをタップ
2. タグ一覧から目的のタグをタップ
3. 該当写真がサムネイル付きで撮影日時降順に並ぶ
4. 写真をタップ → Google フォトの該当ページに遷移（純正アプリがインストールされていれば自動的にそちらで開く）

### アカウント切り替え

画面右上のアバターをタップしてメニューを開く。「アカウント切り替え」で別の Google アカウントを選べる。各アカウントは独立したタグデータを持つ。

### サインアウト

同じメニューから「サインアウト」。次回起動時は再度サインインが必要。

## サムネイルが「未取得」になっているとき

新しく付けたタグの写真がスマホ側でプレースホルダー（🖼）になっているのは、サムネイル base64 のバックフィルが未完了の状態。次の手順で解消する。

1. PC で `https://photos.google.com/` を開く
2. その写真がある日付までスクロール
3. 数秒〜数十秒待つ（バックグラウンドで Drive に保存中）
4. PWA を再起動 → サムネイルが反映される

「サムネイル N 件未取得」の表示が消えるまでがバックフィル中の状態。

## オフライン挙動

- PWA 自体はオフラインでも起動できる（Service Worker がシェルをキャッシュ）
- タグデータは前回起動時のキャッシュが見えることがある
- サムネイルは Drive 経由で読み込むためオンライン必須
- Google フォト本体（写真をタップして実物表示）はオンライン必須

## トラブルシューティング

| 症状 | 対処 |
|---|---|
| サインインで `redirect_uri_mismatch` | Cloud Console の Authorized JavaScript origins に PWA の URL（origin のみ）が登録されているか確認 |
| サインインしてもすぐ画面が空白 | F12 → Console でエラー確認。多くは `config.js` の不正か、Service Worker の古いキャッシュ |
| 画像が全部プレースホルダー | バックフィル未完了。上記「サムネイルが未取得」の手順 |
| Google フォトアプリが起動せず Web で開く | OS 設定で `photos.google.com` を「常に Google フォトアプリで開く」に設定（Android） |
| 「サインイン」しか出ない | OAuth トークン期限切れ。サインアウト → 再サインイン |
| 別アカウントに切り替えてもタグが見えない | そのアカウントでまだタグ付けしていない＝Drive にデータが無い。各アカウントは独立 |
